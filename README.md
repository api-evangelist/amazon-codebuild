# Amazon CodeBuild API

AWS CodeBuild is a fully managed continuous integration service that compiles source code, runs tests, and produces ready-to-deploy software packages, eliminating the need to provision, manage, and scale your own build servers.

## APIs

- **Amazon CodeBuild API** - API for managing continuous integration builds in the cloud. Supports creating and configuring build projects, starting and stopping builds, viewing build logs and history, managing build environments, and integrating with source providers like CodeCommit, GitHub, Bitbucket, and S3.
